Numerical Method Library for finding zeros for functions, solving equations systems and curve aproximation through linear regression.

Developed using a Class structure for each field and method respectively. The goal was to create an accessible library through Foreign Function Interface, so the library is compiled as Dynamic Link Library (on Windows) and can be processed and show its results in visualizing tools such as Mathplotlib(Python) or Flchart(Dart/FLutter).

In order to compile you must use de CMAkeList.txt that generates the .dll file.
